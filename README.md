# Generative_AI_Research_33077055

**README.md**

**Purpose:** This repository is for files for the study on "Using Generative AI Tools - Boon or Bane."

**Navigate files:** To find desired file, navigate to the appropriate folder containing the content and then following the naming convention find the file.

**How to Contribute:** Place content in appropraite folder and use naming conventions as per below.

**Outline/Folder Structure:**

- lit-review: Literature Review content 
- quantitiave-analysis: Quantitative Analysis content 

- qualitative-analysis: Qualitative Analysis content 

- reports-and-drafts: Drafts and Reports content 

- other: Additional Materials 


**Naming conventions for files:**

- Use hyphen to seperate metadata in file name and spaces between words 

- For version use date in ISO-8601 date format: YYYY-MM-DD 


1. **Literature Review Files** 

   Journal article:  journal-[journal-name]-[article-title].[file-type-extension] 

   Conference article: conf-[conference-name]-[article-title].[file-type-extension] 

   Newspaper article : news-[article-title]-[version].[file-type-extension] 
   Book: book-[title-of-book].[file-type-extension] 


2. **Quantitative Analysis Files**  

   Analysis files:  [analysis-description].[file-type-extension] 

   Survey questions: questions-[survey-name]-[][version].[file-type-extension] 

   Survey data: data-[survey-name]-[version].[file-type-extension] 

   Survey analysis report: report-[survey-name]-[version].[file-type-extension] 


3. **Qualitative Analysis Files**

   interview transcripts: iview-transcript-[version].[file-type-extension] 

   interview protocols: iview-protocols-[version].[file-type-extension] 

   Consent forms: consent-forms-[version].[file-type-extension] 

   Analysis-insights report: analysis-[version].[file-type-extension] 


4. **Drafts and Reports Files**
   Draft research proposals: draft-proposal-[title]-[version].[file-type-extension] 

   Conference papers: conf-[version].[file-type-extension] 

   Final reports: final-report.[file-type-extension] 


5. **Other**
   media files: [media-type]-id[id number].[file-type-extension] 

   photos: photo-[id number]-[version].[file-type-extension] 

   info sheets: infosheet-[title]-[version].[file-type-extension]
